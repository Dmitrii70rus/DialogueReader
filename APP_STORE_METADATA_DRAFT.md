## Description
DialogueReader is a lightweight dialogue voice studio for creators, learners, and anyone who wants to hear text spoken by multiple voices.

- Paste or type text into a clean editor.
- Split dialogue by line into smart segments.
- Assign each segment to one of four speakers.
- Choose a unique system voice for each speaker.
- Play a single segment or run the full dialogue in sequence.

### Free vs Premium
Free users can run up to 3 full-dialogue playback sessions after segmentation.

Unlock **DialogueReader Premium** (one-time purchase) to remove the limit.
